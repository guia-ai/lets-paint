```
   │ ████ ▓▓▓▓ ░░░░  ← 28 colores de la época     │
   └──────────────────────────────────────────────┘
```

- **16 herramientas de verdad**: hasta la curva, el polígono y el aerógrafo.
- **Clic izquierdo** pinta con el color 1 · **clic derecho** con el color 2.
- `Ctrl+Z` deshacer · `Ctrl+S` guardar PNG · `Supr` borra la selección.
- Menú **Imagen** → rotar, voltear, invertir colores, estirar… todo está.
- ¿Texto? Herramienta **A**, arrastrá un cuadro, escribí, clic afuera.

- **🔥 el mechero**: arrastrá para quemar de a un bloque. Pero cuidado dónde ponés el fuego. *A veces te ayuda. A veces te vuela el trabajo.*

## 🎨 ATELIER — pintar en calma

```
        ✦ las herramientas flotan como gotitas ✦

      🖌️  ✒️        ┌─────────────────────────┐
      👆  🩹        │                         │
      💧  🎯        │      (papel crema,      │
                    │    suave como hoja      │
     Tamaño ──○──   │       de verdad)        │
     Carga  ───○─   │                         │
                    │                         │
     ● ● ● ● ● ●    └─────────────────────────┘
     ● ● ● ● ● ●        
     paleta de           💛 apoyá   💬 contame
     ensueño
```

| Herramienta | Qué hace |
|---|---|
| 🖌️ **Acuarela** | se funde y construye color en capas (bajá la *carga* para veladuras) |
| ✒️ **Tinta** | línea limpia para contornos |
| 👆 **Difuminar** | corré el color con el dedo, como pintura húmeda |
| 🩹 **Goma** | borra suave, hacia el papel |
| 💧 **Aguada** | inundá un área — *fluye por debajo de tus trazos* 😍 |
| 🎯 **Cuentagotas** | robá un color del lienzo |

**El truco de oro:** pintá tus figuras primero y tirá la aguada del fondo AL FINAL — el color abraza los trazos sin taparlos. Magia de acuarela real.

**Atajos** (para no soltar el mouse): `B` acuarela · `I` tinta · `S` difuminar · `E` goma · `F` aguada · `1-9` colores · `[` `]` tamaño del pincel · `Ctrl+Z/Y`

**Y un secreto a voces:** todo lo que flota se puede **agarrar y mover**. Armá tu mesa de trabajo como quieras — queda guardada. ¿La despeinaste? **↺ Reordenar**.
